Dataset Requirements

  • Data are in the smart cities domain
    • They belong to the Connecticut Department of Energy & Environmental Protection and are about electric car charging stations.
  • Data are available as a CSV file
  • Data have an open license so it can be published.
    • On the site is specified that the dataset is Open Source.
    • Their license is https://portal.ct.gov/DEEP/Air/Mobile-Sources/EVConnecticut/EVConnecticut---Home
  • Data can be easily linked with generic real-world entities (e.g., locations)
    • The data contains charging stations addresses. These can be linked with coordinates via Wikidata.
    • By leveraging their geospatial coordinates, we can link these stations to nearby entities, landmarks, or points of interest available on Wikidata.
  • Documentation exists for the data (optional)
    • In the site is not present, but we've contacted the owner of the dataset to ask for a documentation, if provided we will upload it.
  • Data can come from multiple data sources (optional)
      https://data.ct.gov/Transportation/Electric-Vehicle-Charging-Stations/jfhb-ebu6
