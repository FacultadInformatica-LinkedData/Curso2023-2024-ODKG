Moby Bikes bikeshare in Dublin App Requirements

App requirement:

This is the applicatiorequirements for Moby Bikeshare app based on "the Moby Bikes bikeshare dataset"

Main Objective: Display the bike usage with all their features for Mobby bike analysts in Dublin

  • R1: To display a interactive dashboard for the users to use
  • R2: To display bike usage pattern for the users on daily, weekly or monthly timeframe (Our dataset limits us to these options)
  • R3: To show graphs and charts for:
      Most frequently used bike
      How much battery left for each bike on average
      Display information about each bike

Mock-up:

Dashboard: A dashboard that has options for the wanted time frame the user wishes to select, below it the graph options that the user wishes to display, displaying charts and graphs for analysis
