Moby Bikes bikeshare Dataset Requirements:

This is the dataset requirement checklist for Moby Bikeshare dataset found in Smartdublin open datastore

  • R1: Data are in the smart cities domain (E-Bike Tracking).
  • R2: Data are available as a CSV file.
  • R3: Data have an open license so it can be published (Creative Commons Attribution License (cc-by)).
  • R4: Data can be easily linked with generic real-world entities (e.g., locations).
