Divvy Trips Dataset Requirements:

This is the dataset requirement checklist for Divvy Trips dataset found in Kaggle

  • R1: Data are in the smart cities domain (Bike Tracking).
  • R2: Data are available as a CSV file.
  • R3: Data have an open license so it can be published (Divvy Data License Agreement).
  • R4: Data can be easily linked with generic real-world entities (e.g., locations).
