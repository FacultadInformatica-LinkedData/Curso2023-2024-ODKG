Analysis Dataset Group02.

Requirements of MadridAlFresco

(The data will be used to create a guide to places to find terraces).

Data Analysis:

Name Type Description
id_terraza Integer Unique identifier for the terrace.
id_local Integer Unique identifier for the establishment.
id_distrito Integer Unique identifier for the district.
nombreDistrito String Name of the district.
barrio String Name of the neighborhood.
tipoVia String Type of road (street, avenue, etc.).
nombreCalle String Name of the street.
portal Integer Number of the portal.
codPostal Integer Postal code.
coordenadaX Float X coordinate for location on a map.
coordenadaY Float Y coordinate for location on a map.
situacion String Description of the establishment's situation (open, closed, etc.).
rotulo String Label of the establishment.
superficie Float Area of the establishment in square meters.
horaApertura Time Opening time of the establishment.
horaCierre Time Closing time of the establishment.
mesas Integer Number of tables on the terrace.
sillas Integer Number of chairs on the terrace.

Data Resources:

Name Type Description
Distrito Resource Contains the neighborhood, street type, street name, the postal code, and portal where a certain establishment is located.
Local Resource Contains the label of the establishment as a string, the coordinates (x,y) to locate the establishment on the map, the opening and the closing time.
Terraza Resource Contains a float representing the surface area, and two integers for the number of tables and chairs allowed, respectively.
