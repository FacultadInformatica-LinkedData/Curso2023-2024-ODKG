Datasets analysis



US Colleges and Universities

The path to the dataset can be found HERE

This dataset provides all Post Secondary Education facilities such as Doctoral/Research Universities, Masters Colleges and Universities, Baccalaureate Colleges and more.

This dataset contains 6559 instances with the variables chosen for our dataset being:

  • IPEDSID (PK): The dataset includes the Integrated Postsecondary Education Data System's ID number making it possible to easily merge them with our other datasets. As key, there are no missing values and 6474 unique values.
  • Name (String): This column provides the names of the universities. There are 6349 different universities or colleges and there are no missing values.
  • Description (String): This column gives the North American Industry Classification System description for the institution. There are 9 possible descriptions applicable and every institution has one."COLLEGES,UNIVERSITIES,AND PROFESSIONAL SCHOOLS" makes up to 39% of the descriptions followed by "JUNIOR COLLEGES" with 24%.
  • Website (String): This column provides the url of the institution. There are 118 missing values.
  • Address (String): This column provides the street where the institution is located. There are 12 missing values and 6394 unique values since some adresses have the name of the campus where more than one institution can be located.
  • Latitude (Float) : This column provides the latitude coordinate of the location of the institution.Since this dataset contains primarly north american institutions, the most common values are found in the interval [24,49] having a 97% of apperance in the dataset.
  • Longitud (Float) : This column provides the longitud coordinate of the location of the institution. As well as the latitide, in the [-130,-60] interval you can find 99% of the values.
  • City (String) : This column provides the name of the city where the institution is located. There are 2351 unique cities where these institutions are located.
  • State (String) : This column provides the name of the state where the institution is located. There are 59 unique states where these institutions are located. The state with the majority of institutions in this dataset is "California" with 11% followed by "New York" and "Texas" with 6%.
  • County (String) : This column provides the name of the counties where the institutions are located. There are 1409 unique counties where these institutions are located being "Los Angeles" with the most institutions having 3% of them followed by "COOK" and "ORANGE" with 2%.


US University & College Rankings

The path to the dataset can be found HERE

This dataset provides the rankings for the top 150 US universities and top 250 colleges since 1983.

This dataset contains 528 instances with some of the data being incomplete, specially in the earlier years. The variables chosen for our dataset are:

  • IPEDSID (PK): As mentioned before, this column contains the ID Number to merge with the other datasets. There are no missing values and each value is unique.
  • Years(String): There are 38 different columns, with their name being the year in which the ranking was made, containing the positions in which the university or college ended up. This column is a string instead of an integer because for the college rankings, the colleges ranked below 100, depending of the year, are ranked in intervals such as "T3 (81-120)" or "T4 (121-160)". The most recent years don't have almost any missing values but, for example, in the universities ranking, before 1996, there are a 85% of missing values.


American University Data

The path to the dataset can be found HERE

This dataset provides the enrollment and graduation rate for most american institutions.

This dataset contains 1534 instances and the variables we have chosen for our dataset are:

  • ID Number (PK): As every dataset we have chosen, this column contains the IPEDSID number to merge with the other datasets. There are no missing values with each value being unique.
  • Applicants total (Integer): This column provides the number of applicants in a university per year. The dataset is missing 10% of the values and the mean of applicants is 6391 while the values range from 0 to 72676.
  • Enrolled total (Integer): This column provides the number of enrolled students in a university per year. The dataset is missing 10% of the values and the mean of enrolled students is 1043 while the values range from 0 to 10241.
  • Admissions total (Integer): This column provides the number of admited students in a university per year. The dataset is missing 10% of the values and the mean of enrolled students is 3554 while the values range from 0 to 35815.
  • Graduation rate (Integer): There are 3 different columns, where each one refers to the graduation rate depending on the years of completion of the degree (4,5 or 6). These columns have the same number of missing values which is 4% and their values range from 0 to 100 having an average of 47 graduates per university per year.
