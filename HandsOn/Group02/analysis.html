Requirements of MadridAlFresco

(The data will be used to create a guide to places to find terraces).

Data Analysis:

Distrito String Contains the neighborhood, street type, street name, the postal code, and portal where a certain establishment is located.
Local String Contains the label of the establishment as a string, the coordinates (x,y) to locate the establishment on the map, the opening and the closing time.
Terraza String Contains a float representing the surface area, and two integers for the number of tables and chairs allowed, respectively.
