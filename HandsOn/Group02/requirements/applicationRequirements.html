Application Requirements

MadridAlFresco Requirements

(The data will be used to create a guide to places to find terraces)

R1. Map and Search Box

The application must allow users to visualize a map and a search box.

R2. Location Input

The search box should enable users to enter a location, and upon submission, the map should automatically center on that location, displaying restaurant hotspots.

R3. Information Display

For each restaurant hotspot, the application should present information about the place in a standardized format across all centers.
