Requirements for Selected Datasets

This HTML document discusses whether the selected datasets satisfy the following requirements:

R1. Data are in the smart cities domain.

Yes, the information is provided by the Madrid City Council (Ayuntamiento de Madrid).

R2. Data are available as a CSV file.

The documents are available in CSV format.

R3. Data have an open license so it can be published.

According to the conditions on the website: Madrid Open Data Portal, documents such as CSVs should include the last update date and the information source, which is "Ayuntamiento de Madrid" (Madrid City Council).

R4. Data can be easily linked with generic real-world entities (e.g., locations).

The information is provided in Spanish with simple CSV attribute names.

R5. Documentation exists for the data (optional).

Documentation types include "Dataset Structure (PDF, 21 Kbytes)" and "Identifiers for Municipal Wi-Fi Networks (XLSX, 41 Kbytes)."

R6. Data can come from multiple data sources (optional).

Multiple CSVs have been included, created in different projects by the city council.
