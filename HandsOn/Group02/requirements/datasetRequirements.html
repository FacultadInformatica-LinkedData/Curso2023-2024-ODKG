List of Requirements

Requirements Description
R1. Data is in one of the selected smart city domains. Yes, the datasets belong to the domain of one of the Smart Cities (Madrid).
R2. Data is available as a CSV file. Yes, the selected datasets are available in the form of CSV files.
R3. Data has an open license so it can be published. Yes, these datasets have a free license that allows the reuse of these documents for various commercial and non-commercial purposes.
R4. Data can be easily linked with generic real-world entities (e.g., locations). Yes, this data can be linked to various generic entities in the world, such as nightlife venues, personal and auxiliary services, squares, water sources, etc.
R5. Documentation exists for the data. Yes, everything is properly documented.
R6. Data can come from multiple data sources. No, the extracted information come from one source (Ayuntamiento Madrid), but can be linked to Wikidata.
