Madrid Waste Management Analytics: Dataset Requirements

This document aims to assess if our dataset meets the specific requirements needed for our project. The following checklist provides a summary of each requirement, and a brief description of how our dataset satisfies them.

  • R1. Data are in the smart cities domain: Monitoring and managing waste production is crucial for urban sustainability. Collecting data on waste production, especially when classified by type, can help city planners make informed decisions about recycling programs, waste collection schedules, and other sustainability initiatives.

  • R2. Data are available as a CSV file: The data has been downloaded in a CSV format from The Official Portal for European Data.

  • R3. Data have an open license so it can be published: Their license can be found here.

  • R4. Data can be easily linked with generic real-world entities (e.g., locations): The waste collection data includes location-specific details (neighborhoods, districts) that can be linked with real-world entities.

  • R5. Documentation exists for the data (optional): The official documentation can be found here.

  • R6. Data can come from multiple data sources (optional): There are three datasets for three different years (2020, 2021, 2022) that will be used to provide deeper insights and analyses of waste collection and consumption.
