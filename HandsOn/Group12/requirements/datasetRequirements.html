Dataset Requirements

R1. Data are in one of the smart city domains ✔

    The data are from the city of Madrid: 

    - https://datos.madrid.es/portal/site/egob

    - https://datos.gob.es/es/

R2. Data are available as a CSV file ✔

    Data are available to download as a csv file in the open data web of the city of Madrid and the open data web of the government of Spain.

R3. Data have an open license so they can be published ✔

    The data have an open license that allows to reuse it for commercial and non-commercial purposes. This license can be found here: https://datos.madrid.es/egob/catalogo/aviso-legal

R4. Data can be easily linked with generic real-world entities ✔

    The files contain the names of different districts and streets of Madrid which can be linked to the same locations in other datasets such as wikidata.

R5. Documentation exists for the data ✔

    Each dataset has a web page with additional information:

    - https://datos.madrid.es/sites/v/index.jsp?vgnextoid=7eb840252363c710VgnVCM1000001d4a900aRCRD&vgnextchannel=374512b9ace9f310VgnVCM100000171f5a0aRCRD

    - https://datos.gob.es/es/catalogo/l01280796-aparcamientos-rotacionales-historico-de-ocupacion1

R6. Data can come from multiple data source ✔

    The data come from two different sources, the open data web of the city of Madrid and the open data web of the government of Spain.
