Data Set Analysis

The dataset presented in the corresponding CSV file contains all cultural activities and events included in the Madrid City Council.

Data Set Description

In the dataset, we have a total of 1102 records, meaning 1102 elements. For each of the elements, we have 29 attributes to define them. In some cases, there are missing data, such as in the price, days of the week, or excluded days. However, in general, we have a fairly complete and consistent dataset.

Data Set Properties

  • Number of Records: 1102 cultural events in Madrid
  • Event ID: 1102 elements: Integer
  • Event Title: 1102 elements: String
  • Price Range: From free to 140 euros, with 52 different price values: String
  • Free Events: A total of 216 free events and 885 that are not: Boolean
  • Long-Duration Events: A total of 263 long-duration events and 838 that are not: Boolean
  • Days of the Week: Detailed as L, M, X, J, V, S, D: String
  • Excluded Days of the Week: Detailed as L, M, X, J, V, S, D: String
  • Dates: Events from 01/19/2019 to 01/12/2024: Datetime
  • Hours: Event times: Datetime
  • Description: Data on cultural events in Madrid, including information on prices, schedules, and locations: String
  • Event URLs: With 1101 available event URLs: String
  • Activity URLs: With 276 available activity URLs: String
  • Facility URLs: With 151 available cultural facility URLs: String
  • Accessible Facilities: A total of 11 elements to determine facility accessibility: List(Int)
  • Location: Includes a description of the street class (e.g., "CALLE"), street name, street number, district, neighborhood, postal code, and coordinates: String
  • Audience: Includes information about the type of audience suitable for each event, e.g., /Familias: String
