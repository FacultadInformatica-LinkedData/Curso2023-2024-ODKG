Dataset Requirements - Group 02

Requirement 1 -> Data are in the smart cities domain: We believe that the dataset we have chosen satisfies this requirement because we are working with data from the Madrid City Council.

Requirement 2 -> Data are available as a CSV file: We believe that the dataset we have selected satisfies this requirement because we have downloaded the dataset in CSV format.

Requirement 3 -> Data have an open license so it can be published: All datasets operate under the open license at the following link: License Information

Requirement 4 -> Data can be easily linked with generic real-world entities (e.g., locations): The dataset on Cultural events can be linked to The dataset of Cultural centers where those events may take place. We can link both dataset to locations (districts, streets), cultural center webpages and organizers/participants identifiers if needed as well.

Requirement 5 -> Documentation exists for the data: The dataset is well documented in a pdf file detailing its structure and columns. Documentation for the Data Catalog: Madrid Data Catalog Documentation Documentation for Cultural events datasets: Dataset Documentation

Requirement 6 -> Data can come from multiple data sources: We are considering datasets for both the cultural activities agenda as well as the cultural centers available in the city.
