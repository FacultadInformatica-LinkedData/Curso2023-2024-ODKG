Resource Naming Strategy:

  • We have created two classes "Calle" and "registroPeatones".
  • Each "registroPeatones" class belongs to a "Calle" class (perteneceACalle)
  • Each "Calle" class has 3 types of variables, "Calle" means Street:
    • id_calle: is the unique id that defines the name of the street.
    • distrito: defines from where part of the city is the street.
    • codigoPostal: number assigned to different areas or places.
  • Each "registroPeatones" class has 6 types of variables, "registroPeatones" means pedestrians register:
    • id_registroPeatones: is the unique id that defines the register.
    • latitud: numeric value for the vertical position.
    • longitud: numeric value for the horizontal position.
    • fechaHora: date and hour of the register.
    • observacionesDireccion: comments related to the register.
    • numPeatones: the number of pedestrians during the register.
