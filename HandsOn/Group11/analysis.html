Resource naming strategy

URI Form: slash (we have many resources)

Ontological terms path: http://smartcity.linkeddata.es/tate-museum/ontology
Ontological terms example: http://smartcity.linkeddata.es/tate-museum/ontology/Artwork
Individuals path: http://smartcity.linkeddata.es/tate-museum/resource
Individuals example: http://smartcity.linkeddata.es/tate-museum/resource/Artwork/A00020

Domain and path: http://smartcity.linkeddata.es/tate-museum (domain is under control of UPM)

Pattern for ontology: /ontology/TateArtworks/<className>

Analysis of the Data Source

The publisher of the dataset is Tate that owns the dataset jointly with the National Galleries of Scotland.
Tategallery is licensed under the The Creative Commons CC0 Public Domain licence. This means that there is no need of giving attribution, but the owner of the data asks specifically to do it, that is why we will follow this request.
The potential license that may be used is the same, The Creative Commons CC0 Public Domain.

The potential license of the dataset to be generated: CC0PD

Analysis of the Data Set

A dataset is a CSV file with 21 columns:

Column Expected type Description and analysis
id Number An identifier of the record (unique number between 3 and 129068).
accession_number Text Unique identifier of the artwork set by Tate Museum (alphanumeric, non-empty).
artist Text Name of the artist (represented as <surname>,<name>, non-empty).
artistRole Text Role of the artist (97% of the roles are simply 'artist').
artistId Number An identifier of the artist (a number between 38 and 19232, non-empty)
title Text Title of the artwork (non-empty, at most 330 characters).
dateText Text Date of creation. Some records has extra information such as question mark (probably indicating the uncertainty of the date), 'c.' meaning 'circa', the action such as 'printed', 'exhibited', 'remade', or 'date not known' in some cases.
medium Text It reffers to the technique and method that the artwork has been done. The most common one is Graphite on paper. The number of distinct count of medium is 3413.
creditLine Text This column refers to what happened to a particular work. More than half (54.76%) were accepted by the nation as part of the Turner Bequest 1856.
year Number The column refers to the year from which the artwork comes. The youngest is from 1545 , and the most popular year is 1819 ( almost 5% artworks from this year).
acquisitionYear Number Year when the Tate acquired the artwork. The most artwork was obtained in 1856.
dimensions Text It is information regarding the size of the artwork.
width Number Inherited from the column ¨dimensions¨. Values from 3 to 11960. 5% of values is empty. 26 records are invalid.
height Number Inherited from the column ¨dimensions¨. Values from 6 to 37500. 5% of values is empty. 1 record is invalid (contains string 'mm').
depth Number Inherited from the column ¨dimensions¨. Values from 1 to 18290. 96% of values is empty.
units Text Units for the dimension. Either 'mm' (95%) or empty (5%).
inscription Text Inscription. Either 'date inscribed' (9%) or empty (91%).
thumbnailCopyright Text Information on thumbnail copyright (79% are non-empty).
thumbnailUrl Text URL to the thumbnail of the artwork on Tate portal (84% are non-empty).
url Text URL to the page of the artwork on Tate portal (non-empty).

Some of the columns will be exluded from further analysis as they do not contain valuable information. These are the columns dateText and dimensions, as the data they contain is cleaned and saved into other columns.
