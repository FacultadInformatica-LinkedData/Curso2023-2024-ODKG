Analysis of cultural events during the next 100 days and leisure activities in Madrid.

Data Analysis:

The following entities have been found out from the Analysis done to the datasets:
Event Int It is an integer data type that identifies the event that is going to take place.
Concept String Description of the Event.
Audience String Provides information about the target audience of the Event.
Facility Int Integer that Identifies the museums and leisures centers in Madrid.
Postal Address String Name and number of the event's location.
District String It is a string which provide the name of the disctrict in Madrid where the event takes place.


The Analysis has been performed using the dataset's documentation provided on the download page of each one of it:
  • Museums dataset
  • Cultural centers dataset
  • Events dataset
