Group 01 - Dataset Requirements

  1. Data are in one of the selected smart city domains: HERE ✔
  2. Data are available as a CSV file. ✔
  3. Data have an open license so they can be published: HERE ✔
  4. Data can be easily linked with generic real-world entities (e.g., locations). ✔
    Our dataset will be linked to other knowledge graphs such as Wikidata or DBpedia through the features localización, distrito y calle, tipo de vehículo and Estado meterológico.
  5. Documentation exists for the data (optional). ✔
  6. Data can come from multiple data sources (optional): Yes, the data we intend to work is the 2023 version but, there's versions of this same dataset for previous years we can include in our application if time allows it.
