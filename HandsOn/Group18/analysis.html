Data Analysis

We have decided which of the resource's attributes we will keep for our project.

  • We first cleaned our datasets from rows without location data and modified the drug tests column to standarize the boolean values of the dataset
  • We also merged multiple years in the same document
  • We plan on cleaning more the location variable to make it easier to include all its values into our ontology

  • Classes: Accident, Weather, Person, Vehicle, InjuryType, Address, Coordinates, Road, RoadType, District.
